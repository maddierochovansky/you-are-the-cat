# You Are the Cat

A browser-based choice-driven game where you play as a cat navigating daily life.

## Status
In development. Content design complete. Building the playable version.

## Content
- 4 chapters (Kitchen, Living Room, Bedroom, The Void)
- 571 choices
- 75 random events
- 11 easter eggs

## Files
- `index.html` — game (in progress)
- `reference.html` — full content and probability reference
- `style.css` — styles
- `script.js` — game logic
